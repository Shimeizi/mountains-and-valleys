# 🏔️ Mountains and Valleys

A Python project developed for the **Programming Fundamentals** course.

## 📖 Overview

This project analyzes a rectangular territory composed of horizontal and vertical paths. The intersections of these paths can be either empty or occupied by mountains.

The program implements a set of functions to process the territory and obtain information about mountain chains, valleys and connections between intersections.

## ✨ Features

* 🗺️ Territory representation
* 📍 Adjacent intersection detection
* ⛰️ Mountain chain identification
* 🌿 Valley computation
* 🔗 Connectivity analysis between intersections

## 🛠️ Built With

* 🐍 Python
